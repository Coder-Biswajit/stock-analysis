# 📊 Stock Analysis Web App

A Flask-based web app to analyze stocks using yfinance.

## 🚀 Features
- User Login/Register
- Watchlist
- Stock Charts
- Buy/Sell Signal
- Data Analysis

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- yfinance
- Matplotlib
